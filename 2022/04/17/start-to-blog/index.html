最近又突然有一些想寫 blog 的動機:

黑暗執行緒

我不是第一次看到這個 blog，但是因為上班時有一些空檔時間不知道要做什麼，
滑 Github 的時間都已經算進去了，而很剛好的，部門目前在使用的主力開發語言是使用
C# .Net Framework，黑暗執行緒 剛好有許多的 dotnet 文章，
所以我就時不時地點開來看看。

…
…
…

但是這個跟我想寫 blog 有什麼關係?

對阿到底??

  1. 上班沒時間寫，對阿

大概就是這樣吧

只是想折騰自己學習新東西而已
